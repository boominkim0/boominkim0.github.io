# 개요
안녕하세요. 김부민 입니다.
제가 진행한 프로젝트와 생각을 정리하는 공간입니다.
* 페이지 링크: [boominkim0.github.io](https://boominkim0.github.io)

## 개발
* Nuxt3로 개발했습니다.
* SSG(Static Site Generation)로 빌드해서 Github Pages에 호스팅하고 있습니다.

## 시작
* install
    ```bash
    npm install
    ```
* dev
    ```bash
    npm run dev
    ```
* build
    ```bash
    npm run build
    ```
* deploy
    ```bash
    npm run deploy
    ```
